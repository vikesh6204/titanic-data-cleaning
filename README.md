Titanic Data Cleaning Project
# Overview

This project performs data preprocessing and cleaning on the Titanic dataset.

# Tools
Python
Pandas
NumPy
# Steps
Filled missing values
Age, Fare → median
Embarked → mode
Cabin → "Unknown"
Fixed formatting (lowercase, strip)
Converted Sex → (male=0, female=1)
Removed duplicate rows
Handled outliers using IQR method
# Output

cleaned_titanic.csv
